Covers fundamental Python topics: literals, operators, and if-else statements
Literals are fixed values used in programs such as numeric (10, 3.14), string ("Hello"), boolean (True, False), and None
Operators are used to perform operations on data
Arithmetic operators include +, -, *, /, %, //, **
Comparison operators include ==, !=, >, <, >=, <=
Logical operators include and, or, not
Assignment operators include =, +=, -=
Identity operators include is, is not
Membership operators include in, not in
If-else statements are used for decision making based on conditions
If-elif-else is used to handle multiple conditions
These concepts are essential for building logic and writing Python programs
